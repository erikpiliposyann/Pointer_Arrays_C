# 📊 C Arrays Exercises

This project contains beginner-level **C programming exercises focused on arrays, loops, and basic algorithms**.

The goal is to practice working with:

* different data types arrays
* traversal
* searching
* aggregation
* simple computations

---

# 📌 1. Multiple Data Types Arrays

Declare arrays of different types:

* `int`
* `float`
* `double`
* `char`

### Tasks:

* Input values using `scanf`
* Print all arrays

---

# 📌 2. Reverse Array

Reads an array and prints it in reverse order.

### Example:

```text id="rev"
Input:  1 2 3 4 5  
Output: 5 4 3 2 1
```

---

# 📌 3. Sum of Array Elements

Calculates the sum of all elements in an integer array.

---

# 📌 4. Maximum Element

Finds the largest element in an array.

---

# 📌 5. Minimum Element Index

Finds the index of the smallest element in an array.

---

# 📌 6. Sum of Min and Max

Finds:

* minimum element
* maximum element

Then prints their sum.

---

# 📌 7. Divide Elements by Size

Each element is replaced with:

```text id="div"
arr[i] = arr[i] / n
```

---

# 📌 8. Search Element in Array

Checks whether a number exists in the array.

### Output:

* `YES` → if found
* `NO` → if not found

---

# 📌 9. Count Odd Elements

Counts how many odd numbers are in the array.

---

# 📌 10. Average of Array

Computes arithmetic mean:

```text id="avg"
average = sum / n
```

---

# 🛠️ Concepts Used

* Arrays (1D arrays)
* Loops (`for`)
* Conditional statements
* Searching algorithms
* Aggregation (sum, min, max)
* Type casting
* Input/output (`scanf`, `printf`)

After completing these exercises, you will understand:

* How arrays work in C

* How to implement basic algorithms
* How to manipulate and analyze data in arrays
* How to solve real-world beginner problems using loops
