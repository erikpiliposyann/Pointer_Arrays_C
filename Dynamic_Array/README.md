# ⚙️ C Functions, Recursion & Dynamic Memory Exercises

This project contains intermediate-level **C programming exercises focused on functions, recursion, pointers, and dynamic memory management**.

---

# 📌 1. Sum of Digits

### 🧠 Task

Calculate the sum of digits of a number.

### Example:

```
Input: 123
Output: 6
```

### Concepts:

* Modulus operator `%`
* Integer division
* Looping

---

# 📌 2. Maximum in Dynamic Array

### 🧠 Task

Find the maximum element in a dynamically allocated array.

### Concepts:

* `malloc`
* pointer arrays
* traversal

---

# 📌 3. Modify Array (Index × Value)

### 🧠 Task

Replace each element with:

```
arr[i] = arr[i] * i
```

### Concepts:

* In-place array modification
* indexing
* dynamic memory

---

# 📌 4. Remove Even Elements (Using realloc)

### 🧠 Task

Remove all even numbers from a dynamic array using `realloc`.

### Concepts:

* dynamic memory resizing
* pointer manipulation
* shifting elements
* memory safety

---

# 📌 5. Recursive Reverse Digit Printing

### 🧠 Task

Print digits of a number in reverse order using recursion.

### Example:

```
Input: 1234
Output: 4 3 2 1
```

### Concepts:

* recursion
* base case
* modulo operations

---

# 📌 6. Recursive Array Printing

### 🧠 Task

Print all elements of an array recursively.

### Concepts:

* recursion with pointer arithmetic
* termination condition

---

# 📌 7. Factorial with Memoization (Caching)

### 🧠 Task

Compute factorial using recursion + caching.

### Concepts:

* recursion
* static memory (cache array)
* memoization technique

---

# 📌 8. Concatenate Two Numbers

### 🧠 Task

Join two integers into one number.

### Example:

```
Input: 345, 89
Output: 34589
```

### Concepts:

* digit counting (`log10`)
* `pow()` function
* number manipulation

---

# 🛠️ Concepts Used

* Functions in C
* Recursion
* Dynamic memory (`malloc`, `realloc`, `free`)
* Pointers
* Arrays
* Mathematical operations
* Memoization techniques

After completing these exercises, you will understand:
