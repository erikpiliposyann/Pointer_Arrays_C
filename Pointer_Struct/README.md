# Advanced C Programming Tasks

This folder contains advanced C programming exercises covering recursion, dynamic memory allocation, matrices, and structures.

---

## 📌 Topics Covered

- Recursion
- Dynamic memory allocation (malloc, realloc, free)
- 2D arrays (matrices)
- Structs in C
- Mathematical computations

---

## 🧩 Exercises

### 1. Recursive Power Function

Write a recursive function that calculates num^pow.

Example:
Input: 5, 3  
Output: 125

---

### 2. Remove Alternating Sign Rows from Matrix

Given a matrix of real numbers, remove all rows that represent an alternating sign sequence  
(e.g. positive, negative, positive, negative...).

Each row starts with a positive number.

Requirements:
- Use dynamic memory allocation
- Use malloc and realloc
- Properly free removed rows

---

### 3. Point Structure (2D Geometry)

Create a struct Point with:
- float x
- float y

Implement functions to:
- Input point values
- Calculate distance between two points
- Calculate distance from origin (0,0)
- Find midpoint between two points
